Here's a handy checklist to ensure a smooth import:

  • Employee Id, Customer Name, Currency Code, are mandatory fields.
  • The first row of your file must contain the field names (e.g., Employee Id, Customer Name, Currency Code, Amount, etc.).
  • Please ensure that all customers listed in the Excel file already exist in the system. To ensure a smooth submission, add any missing customers before proceeding with the import.
  • Maximum allowed row limits per import are 5000 rows for .xls, .xlsx, or .vcf and 10,000 rows for files.
  • Your file's size should not exceed 10MB.
